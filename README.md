# 🚀 Шаблон сайта портфолио

Учебный проект — шаблон сайта-портфолио, сверстанный в рамках обучения на курсе в ВебКадеми 🎓

## ✨ О проекте

Это простой и аккуратный шаблон портфолио, созданный в процессе обучения верстке и базовому фронтенду.  
Проект помогает закрепить навыки работы с макетами, структурой страниц и интерактивностью.

## 🛠️ Технологии

- 🧱 HTML — семантическая разметка  
- 🎨 SCSS — стилизация и удобная организация CSS  
- ⚙️ JavaScript — базовая интерактивность  

## 📱 Возможности

- 📐 Адаптивная верстка  
- 🎯 Чистая и понятная структура кода  
- 🎨 Стилизация через SCSS  
- ⚡ Простые интерактивные элементы  
- 🧩 Готовый шаблон для портфолио  

## 🎓 Обучение

Проект выполнен в рамках курса ВебКадеми 💻  
Цель — практика верстки и закрепление базовых навыков фронтенд-разработки.
